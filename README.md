# sistema-de-tarefas-php
Gerenciador de tarefas desenvolvido em PHP puro e MySQL. Funcionalidades: Dashboard com métricas visuais, cadastro de usuários e projetos, controle de status e prazos. Utiliza Bootstrap 5 para o front-end e consultas SQL otimizadas. Perfeito para demonstrar conceitos de CRUD e modelagem de dados.
